Autonomous Tracking Turret

For one of my recent projects, I developed an autonomous water tracking turret that uses machine learning to map and track human body parts. This innovative system is capable of identifying and following human movements, and accurately targeting specific body parts with a stream of water. The project involved integrating computer vision with machine learning algorithms to achieve precise detection and tracking. The turret operates autonomously, making real-time adjustments to track moving targets. This project not only demonstrates my skills in hardware and software integration but also showcases my ability to apply machine learning to create interactive and responsive systems.

Playmakers

Playmakers is a 2D platformer football game I developed using the Godot Engine. In this exciting game, players choose from different character classes, each with unique abilities, and aim to shoot the ball through the uprights. The challenge lies in the game's unique twist: if the ball touches the ground, the game restarts. With its engaging gameplay and strategic elements, Playmakers offers a fresh and fun take on traditional football games, showcasing my skills in game design and development.
